# Web前端学习笔记

## 基本术语解释

- **c**: Client (客户端)
- **b**: Browser (浏览器)
- **s**: Server (服务器)

## 架构类型

### C/S架构 (Client/Server)

- **定义**: 在C/S架构中，客户端和服务器直接进行交互。客户端应用程序直接与服务器进行通信，处理数据和任务。

### B/S架构 (Browser/Server)

- **定义**: 在B/S架构中，客户端通常是通过网络浏览器与服务器进行交互。这种模式更加依赖于Web，用户通过浏览器访问服务器上的资源和服务。
